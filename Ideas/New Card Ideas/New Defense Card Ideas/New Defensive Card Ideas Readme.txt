This is the folder for new defense card ideas. Try to stick to this format:
File name: Card name

File Content:

Card name: the card's name
Card type: support, defense, offense, coin
	-can have "special" super type (eg: special coin, offensive)
Card pick up ability: what happens when a player picks the card up from the map
Card equip ability: triggered based on card type (explained in rules)
Card worth: value of card. only really applies to special cards and coin cards
Card flavor text: a description or quote. it probably should be related to your card
